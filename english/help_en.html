Multi from Camera is a fast way to catalog all the items in a single location. You simply take one photo of each item and when you are done you will have a list of all the items in that location. You can fill in item details or add additional photos at your convenience.

Single from Camera is used to add a single item with multiple photos. Take as any photos of the item as you want and when you are done you will have a single item with all the photos in it.

Single Item adds a new empty item without any photos.

You data lives on your device so it is important to ensure your iOS device is backed up regularly to avoid data loss.

There is an Auto Backup option in the settings screen or you can use also manually backup your data using the Backup option.

Backups are saved in iCloud and can be restored on any other device connected to your Apple ID by using the Restore option in the settings screen.

Backup files can also be accessed using the iOS Files app or using iCloud.com on your computer.

Backup generates smaller files and is faster because the photos are saved to a separate folder in iCloud. These files can only be imported into devices connected to the same Apple ID.

Export generates self-contained backup files with photos included in the export file. This makes export files much larger but the advantage is that they can be shared with someone using a different Apple ID.

Enable the All Photos option which can be found in the Report Options screen.

For high-value items you can generate a single item report from the item screen which includes all the item's photos in a larger format.

This will create a zip file containing all the original photos for the items in the report that will be saved alongside the report in iCloud.

Photo archives are very large so you should consider whether it is necessary before enabling this option.

Photo Archives are not visible in the Saved Reports screen and can only be accessed using the iOS Files app or using iCloud.com on your computer.

Many sync issues can be resolved by disabling syncing on all your devices and then re-enabling it on all your devices.

Note that WiFi syncing will only happen when Home Contents is open and running on both devices at the same time and it may take a minute or two for the devices to fully sync.

Another option is to manually sync your data using the Backup and Restore options. Note that Restore will never delete data and will only add missing data to your device so it can be used to merge data from multiple devices.
